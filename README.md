## CodeLeap Frontend Test

This project is my solution for the CodeLeap frontend engineering test.

The goal of the challenge was to build a simple application that performs basic CRUD operations using the provided API.

---

# Live Demo

Deployed application:

https://your-vercel-link.vercel.app](https://code-leap-frontend-test-beige.vercel.app

---

# Tech Stack

- React
- TypeScript
- Vite
- Styled Components
- Fetch API

---

# Features Implemented

Core requirements:

- User signup (frontend only)
- Create posts
- List posts
- Edit posts
- Delete posts
- Posts sorted by most recent
- Edit and delete only available for the post owner

---

# Bonus Features

Some additional improvements were implemented:

- Responsive layout for mobile devices
- Persistent login using localStorage
- Logout functionality
- Hover effects and subtle UI transitions
- Smooth modal animations
